# System Navigation Lab (Windows & Linux CLI)

## What I Did
Used the command line to navigate directories, locate files, and read file contents in both Windows and Linux environments.

## Tools Used
- Windows Command Prompt
- Linux Terminal

## Key Skills
- Navigating directories (cd)
- Listing files (dir / ls)
- Locating files
- Reading file contents

## What I Learned
Gained confidence using the command line to explore systems and locate files, which is important for troubleshooting and security tasks.

## Why It Matters
These skills are essential for cybersecurity roles, as analysts frequently use the command line to investigate systems, locate files, and analyze activity during incidents.
