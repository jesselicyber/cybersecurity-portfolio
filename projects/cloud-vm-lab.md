# Cloud & Virtual Machine Management Lab

## What I Did
Created and managed virtual machines in a simulated cloud environment and investigated system issues.

## Tools Used
- Cloud console (AWS-like interface)
- Virtual machine manager

## Key Skills
- Creating and configuring virtual machines
- Understanding cloud regions and cost management
- Basic troubleshooting of system errors

## What I Learned
Gained exposure to how cloud environments operate and how virtual machines are deployed, managed, and monitored.

## Why It Matters
Cloud platforms are widely used in modern infrastructure, and understanding how virtual machines and cloud resources work is important for security monitoring and incident response.
